# plot-booking

A simple app to book plot.

__TRY HERE:__  https://plot-booking.herokuapp.com/

__CODE HERE:__ https://github.com/raju4789/plot-booking


## BUILT USING:

React Hooks, Spring Boot, Authorisation using JWT and Spring Security, Spring Data JPA, H2(Inmemory SQL database), REST, HTML, CSS, Material-UI.

## Features

__1.__ Login/Signup.

__2.__ Book/Unbook.

__3.__ Color highlighting of plot booking status.

## Deployment steps:

**1.**   Run mvn clean install -DskipTests

**2.**  Run mvn sprin-boo:run

**3**  Paste http://localhost:8080 and rest follows.
