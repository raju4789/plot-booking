# plot-booking

A simple app to book plot.

__TRY HERE:__ 

## BUILT USING:
React Hooks, Spring Boot, Authorisation using JWT and Spring Security, Spring Data JPA, H2(Inmemory SQL database), REST, HTML, CSS, Material-UI.

## Features

__1.__ Login/Signup
__2.__ Book/Unbook
__3.__ Check status.
